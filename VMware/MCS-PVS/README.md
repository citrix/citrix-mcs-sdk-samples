# MCS + PVS PowerShell Provisioning Scripts for VMware

This folder contains **Citrix Machine Creation Services (MCS) + Citrix Provisioning (PVS)** PowerShell automation samples for **VMware**.

The scripts demonstrate how to:

- Use **MCS PVS integration** to work with PVS farms, sites, stores, and vDisks.
- Create **PVS Provisioning Schemes (ProvSchemes)** for VMware-hosted machines.
- Create **Machine Catalogs** backed by PVS using MCS.
- **Remove PVS catalogs** including all VMs, AD accounts, and associated objects.

Typical scripts in this folder include examples for:

- **Local Active Directory-joined** PVS catalogs.
- **Hybrid Entra joined** PVS catalogs.
- **Complete catalog removal** (including VMs).

Use these scripts as reference automation for end-to-end **Citrix MCS + PVS provisioning** scenarios on VMware.
