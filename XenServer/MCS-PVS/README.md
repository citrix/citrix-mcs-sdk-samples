# MCS + PVS PowerShell Provisioning Scripts for XenServer

This folder contains **Citrix Machine Creation Services (MCS) + Citrix Provisioning (PVS)** PowerShell automation samples for **XenServer**.

The scripts demonstrate how to:

- Use **MCS PVS integration** to work with PVS farms, sites, stores, and vDisks.
- Create **PVS Provisioning Schemes (ProvSchemes)** for XenServer-hosted machines.
- Create **Machine Catalogs** backed by PVS using MCS.

Typical scripts in this folder include examples for:

- **Local Active Directory-joined** PVS catalogs.

Use these scripts as reference automation for end-to-end **Citrix MCS + PVS provisioning** scenarios on XenServer.
